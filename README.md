# StreamPy

A Python project demonstrating **OOP**, **functional programming**, **concurrency**, and **system operations**.

##  Description

This project contains Python modules for:
- Media playback (`media_player.py`)
- Movie recommendations (`recommendations.py`)
- Server logging and collections (`server_logs.py`)
- Database operations and concurrency (`data_engine.py`)
- File, network, and system tasks (`network_ops.py`)

##  How to Run

Run any module individually

python src/media_player.py

python src/recommendations.py

python src/server_logs.py

python src/data_engine.py

python src/network_ops.py
